铁磁体中的磁矩运动方程

晶格的磁性结构使其能谱出现特殊的分支，在研究这些能谱时，我们首先回忆磁性物质中相互作用的一些特性。

铁磁体内相互作用的基本形式是原子的交换作用，由其导致自发磁化。这种相互作用的特性在于它与磁化对晶格的取向无关：在考虑到系统波函数的对称性，交换相互作用是电子静电相互作用的结果，并与总自旋的方向无关。

最简单的铁磁系统是晶格中的原子具有磁矩的电介质，并且磁矩平行时恰好在能量上是有利的，以此来确定交换相互作用的符号（“铁磁化”）；这时，系统基态是所有的自旋平行的状态。确切地说，处于基态时系统的总自旋在某一方向的投影等于最大的可能值 (\sum s_a)（对所有原子求和），其中 (s_a) 是一个原子的自旋。事实上，交换作用的哈密顿量 (\hat{H}_{\text{交换}}) 与系统的总自旋算符 (\hat{S}) 可对易，也就是与投影算符 (\hat{S}z) 可对易（因为 (\hat{H}{\text{交换}}) 与各个自旋的方向无关，而算符 (\hat{S}) 是自旋空间中的转动算符）。因此基态应有确定的 (S_z) 值，而能量的最小值对应于 (S_z) 的最大值。我们指出，这时每个原子自旋的投影 (s_z) 等于其最大值 (s_a)，因此基态的磁矩等于自身的“额面”值 (\sum \mu_a)，其中 (\mu_a) 是一个原子的磁矩。但是，这个性质被更弱的、相对论性相互作用所破坏。

复杂一些的情况是物体的磁化强度不等于额面值。尤其，不是所有原子间的相互作用都是铁磁性的时候，可能由两个相反磁化的亚晶格形成结构，亚晶格磁化强度不同，因此不能完全抵消；具有这种结构的物质称为：

  • 铁氧体（完全抵消的情况，称为反铁磁体）。

在研究铁磁性金属的原子自旋时，必然涉及传导电子。传导电子无论如何，甚至在 ( T = 0 ) 时，也不可能完全磁化（由于费米简并效应）。磁性相互作用的特性还使铁磁体具有更复杂的基态结构：磁矩原子的非共线分布——螺旋结构。

与所有宏观系统一样，铁磁体的弱激发态可以看成元激发的集合，即准粒子气体。原子磁矩有序分布中的元激发称为自旋波量子。既然所讨论的是平移对称晶格中的准粒子，自旋波量子具有的动量就不是真实的动量，而是准动量，准动量取遍一个倒格胞中的所有值。在经典图像中，自旋波量子对应于自旋波，即磁矩振动沿晶格的传播。自旋波量子遵从玻色统计，因此，自旋波量子态大的占有数对应于自旋波的经典极限情况。

若自旋波的波长大于晶格常数 ( a )（即波矢 ( k \ll 1/a )），则此波应视为宏观的。此时，波的色散律 ( \omega(k) ) 将通过磁矩宏观运动方程中的唯象参数（物质常数）来表达，因此自旋波量子的谱 ( \varepsilon = \hbar\omega(k) ) 也将通过这些参数确定。确定自旋波量子谱的方法，完全类似于利用声波振动宏观方程中的宏观参数（弹性模量）来确定长波声子谱。为实现这一设想，必需预先导出上述运动方程①。

交换相互作用下的磁矩运动方程

从只考虑交换相互作用的情况开始研究。

我们关注铁磁体的弱激发态（只有弱激发态的性质可用普通形式解释），因此应限于磁矩的低频“慢”运动。磁矩方向在空间缓慢改变而磁矩大小保持不变的运动即属于此类。事实上，磁化强度的平衡值由交换相互作用确定，因此磁化强度的变化在任何波长均不与有限能耗相联系（假设物体处于远离居里点的状态，在居里点自发磁化强度消失）。另一方面，若仅计及交换相互作用，物体作为整体磁矩转动时能量不变；因此波长越大，磁化强度非均匀转动所需的能量越小。换句话说，长波振动的频率低。

决定磁矩方向变化的运动方程为力矩 ( \mathbf{K} ) 作用下自旋角动量 ( \hbar\mathbf{S} ) 的进动方程：

[
\frac{\partial (\hbar\mathbf{S})}{\partial t} = \mathbf{K},
]

（见第一卷（34.4）），其中 ( \hbar\mathbf{S} ) 和 ( \mathbf{K} ) 分别是单位体积的角动量和力矩。磁矩密度（磁化强度）( \mathbf{M} ) 通过 ( \mathbf{S} ) 依下式表达：

[
M = \gamma h S = - \frac{g | e |}{2mc} S,
]

式中 (\gamma) 是磁矩与角动量之比。(e = -|e|) 和 (m) 分别是电子的电荷和质量。(g) 为铁磁体的旋磁比（对比第二卷 §45）。因此，(M) 的方程如下：

[
\frac{\partial M}{\partial t} = \gamma K. \tag{69.1}
]

如力矩 (K) 不依赖坐标，此方程为精确的；如 (K) 在空间缓慢变化，此方程为近似的。

力矩 (K) 定义为自旋系统在无穷小转动时能量的变化。令 (\delta \phi) 为沿旋转轴方向的矢量，其值等于转角，根据力学公式，力矩由 (\frac{\partial E}{\partial \phi}) 给出（见第一卷 (34.6)）。若 (\delta E) 与坐标有关，应将空间每点的微分换成变分。此变分需在温度、体积及物体内每点磁场恒定的情况下进行，对应的热力学势为自由能 (\tilde{F}^0)。于是，力矩密度由以下方程确定：

[
\delta \tilde{F} = -\int K \cdot \delta \phi , dV. \tag{69.2}
]

磁矩方向变化时（磁矩值固定），自由能的变化 (\delta \tilde{F}) 可表示为：

[
\delta \tilde{F} = -\int H_{\text{有效}} \cdot \delta M , dV, \tag{69.3}
]

此处类比外磁场内磁矩能的表达式，引入“有效”场 (H_{\text{有效}})。平衡时 (H_{\text{有效}} = 0)，因磁矩的平衡分布由自由能的极小条件决定。考虑无穷小转动 (\delta \phi) 时，(M) 的变化为：

[
\delta M = \delta \phi \times M.
]

将此式代入 (69.3) 式并对比 (69.2) 式，得出力矩密度：

[
K = M \times H_{\text{有效}}.
]

代入 (69.1) 式，最终得到磁矩的朗道-栗弗席兹方程：

[
\frac{\partial M}{\partial t} = \frac{g | e |}{2mc} H_{\text{有效}} \times M. \tag{69.4}
]

进一步证明，此方程描述的磁矩变化不伴随能耗。能耗计算为：

[
Q = T \frac{\partial S}{\partial t} = -\frac{\partial R_{\text{min}}}{\partial t} = -\frac{\partial \tilde{F}}{\partial t},
]
