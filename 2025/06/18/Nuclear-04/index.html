核反应守恒定律

核反应必须满足：

  1. 电荷守恒：反应前后总电荷量不变
  2. 能量守恒：包括质量能转换 E=mc2E=mc^2E=mc2
  3. 动量守恒：系统总动量不变
  4. 角动量守恒：系统总角动量不变
  5. 重子数守恒：质子、中子总数不变
  6. 轻子数守恒：电子、中微子等轻子总数不变
  7. 宇称守恒：近似成立（弱相互作用中可能破缺）

反应能（Q值）

定义

反应式：x+A靶核→B+yx + \underset{\text{靶核}}{A} \rightarrow B + yx+靶核A​→B+y
反应能：Q=ΔMc2=[Mx+MA−MB−My]c2Q = \Delta M c^2 = \left[ M_x + M_A - M_B - M_y \right] c^2Q=ΔMc2=[Mx​+MA​−MB​−My​]c2

  • Q>0Q > 0Q>0 ：放能反应（释放能量）
  • Q<0Q < 0Q<0 ：吸能反应（吸收能量）

物理意义

  • 动能增量：Q=Ey+EB−Ex−EAQ = E_y + E_B - E_x - E_AQ=Ey​+EB​−Ex​−EA​
  • 结合能增量：Q=By+BB−Bx−BAQ = B_y + B_B - B_x - B_AQ=By​+BB​−Bx​−BA​

核反应截面（Cross-Section）

定义

描述核反应发生概率的有效面积：
σ=RI=NIρ\sigma = \frac{R}{I} = \frac{N}{I \rho}σ=IR​=IρN​
其中：

  • RRR：单位时间单个核发生反应次数
  • III：入射粒子通量（单位时间单位面积粒子数）
  • NNN：单位时间单位体积反应次数
  • ρ\rhoρ：靶核密度（单位体积核数）

单位

靶恩（barn）：1 b=10−24 cm21 \, \text{b} = 10^{-24} \, \text{cm}^21b=10−24cm2

微分截面（Differential Cross-Section）

描述反应产物在空间不同方向的概率分布：

σ(θ,ϕ)=dσdΩ=dnnNsdΩ\sigma(\theta,\phi) = \frac{d\sigma}{d\Omega} = \frac{d n}{n N_s d\Omega} σ(θ,ϕ)=dΩdσ​=nNs​dΩdn​

总截面：σ=∫σ(θ,ϕ)dΩ\sigma = \int \sigma(\theta,\phi) d\Omegaσ=∫σ(θ,ϕ)dΩ

核反应类型

1. 复合核反应（Compound Nuclear Reactions）

入射粒子被靶核吸收形成激发态复合核，随后衰变：

39K+d→41Ca∗→40K+p{}^{39}K + d \rightarrow {}^{41}Ca^* \rightarrow {}^{40}K + p 39K+d→41Ca∗→40K+p

特征：反应时间极短（10−19−10−1310^{-19} - 10^{-13}10−19−10−13秒）

2. 剥离反应（Stripping Reaction）

入射核的一部分被靶核剥离：
d+16O→p+17Od + {}^{16}O \rightarrow p + {}^{17}Od+16O→p+17O

3. 电荷交换反应（Charge Exchange）

入射粒子与靶核交换电荷：

23He+ZAX→13H+Z+1AX{}_{2}^{3}He + {}_{Z}^{A}X \rightarrow {}_{1}^{3}H + {}_{Z+1}^{A}X 23​He+ZA​X→13​H+Z+1A​X

（质子转变为中子）

4. 光核反应（Photonuclear Reaction）

12H+γ→11H+n{}_{1}^{2}H + \gamma \rightarrow {}_{1}^{1}H + n 12​H+γ→11​H+n
